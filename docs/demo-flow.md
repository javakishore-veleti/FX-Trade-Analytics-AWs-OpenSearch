# Demo Flow

## 🎯 Objective

Demonstrate how OpenSearch cross-region UI enables querying data across multiple AWS regions from a single dashboard.

## 🪜 Steps

1. Generate FX trades using simulator
2. Publish events to Kafka
3. Consume and index into OpenSearch domains in different regions
4. Open a single OpenSearch UI
5. Query and visualize data across all regions

## 🎬 Demo Highlight

"Query multiple regions from a single OpenSearch dashboard without data replication or endpoint switching."
